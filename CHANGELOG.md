# Changelog

## v0.1.0

Initial public preview release.

- Add QR pairing with the Airych app.
- Add VioStation and VioCam discovery.
- Add Home Assistant entities for status, detection, snapshots, and camera live view.
- Add native WebRTC playback through the VioStation hub.
- Add cloud snapshot support.
- Add options for device reselection and snapshot refresh interval.
- Add Airych automation service definitions.
